## üöÄ Overview
UrbanKick is a modern, responsive multi-page web application built to showcase a futuristic fashion brand. It features a unique "Gen-Z" aesthetic with neon accents, glitch animations, and a seamless purchase flow.

## ‚ú® Key Features
*   **Cyberpunk Aesthetic**: Dark mode, neon lime/pink accents, and glitch text effects.
*   **Multi-Page Architecture**:
    *   **Home**: "Fresh Drops" ticker, hero parallax, and featured items.
    *   **Collection**: Filterable product catalog.
    *   **Product Details**: Dynamic routing (`?id=x`) with split-layout visuals.
    *   **About**: Brand manifesto with city-based visual grid.
    *   **Success**: Simulated checkout with confetti celebration.
*   **Interactive UI**:
    *   Custom "follower" cursor on desktop.
    *   Mobile-responsive "Burger" navigation.
    *   3D Tilt hover effects on product cards.
*   **Tech Stack**: HTML5, Vanilla CSS3 (Custom Variables), JavaScript (ES6+), Vite.
